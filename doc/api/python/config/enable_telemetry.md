# enable_telemetry

***def enable_telemetry(self) -> None*** 

Enable 'usage data and errors' to be sent to Clicknium online service.

**Parameters:**  
    &emsp;None

**Returns:**  
    &emsp;None

**Example:**
***
```python
from clicknium import clicknium as cc

# enable 'usage data and errors' to be sent to Clicknium online service
cc.config.enable_telemetry()

```
